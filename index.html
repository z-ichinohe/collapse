Collapse Analyzerへようこそ

　このサイトでは年倒壊確率を算出するためなどに用いられる漸増動的解析のフローを、耐力低下を考慮することのできる Ibarra-Medina-Krawinklerモデルを用いた一自由度系の振動解析によって追うことが出来ます。 初めての方は上のボタンをクリックして、誘導付きでお進みください。下のボタンをクリックすると、誘導なしでの閲覧が可能です。




Description

　地震時の建物の挙動を把握する方法の一つに計算機による地震応答解析があります。 この解析は複雑にしようと思えばいくらでも複雑にすることが出来ますが、今回は簡単に建物全体を一つのバネとみなす 一自由度解析をすることにしましょう。

　画面の右側には入力されている地震波形が、左側には地震波を受けて揺れるバネが 描かれています。入力倍率*を0.6倍としているので大きく揺れることもありますが、崩壊には至らないはずです。

60秒かかるので飽きてきたらStopを押して止めてください。もう一度見たい方はRe Runをどうぞ。毎回入力される地震波が変わり、 建物の揺れ方も変わるはずです。

*)ここで用いられている地震波は限界耐力計算の第二種地盤における目標スペクトルにランダム位相、もしくは実地震動位相で適合させた模擬地震動です。

Next:復元力特性

　さて、建物全体をひとつのバネとみなした地震応答解析をご覧いただきましたが、このバネの復元力特性についてはまだお話していませんでした。 バネとは基本的に入力された変位に対して、その変位を戻そうとする向きに復元力を発生させますが、微小変位においては変位と力が線形の関係にあるものの 大地震動を受けて変位が拡大すると、その線形の関係が崩れ、段々と複雑な応答を示すようになります。次はこの複雑で多彩な復元力特性に進むことにしますが、 まずは先ほどの一自由度系振動解析における変位ー力関係をアニメーションでご覧いただきましょう。

　復元力特性は履歴的な変位の入力と復元力が一対一に対応させる関係です。今回は比較的単純なバイリニア型のCloughモデルと、 耐力低下を考慮することのできるIbarra-Medina-Krawinklerモデル、さらに耐力低下を単純にした代わりに骨格曲線を任意にした オールマイティな復元力特性の3つを取り上げています。復元力特性で重要なのはMonotonicな載荷で現れる骨格曲線の概形と、 再載荷時の指向点を決める規則と、Cyclicな載荷で現れる耐力低下の3つです。

　骨格曲線は降伏のみを考慮するバイリニア、もしくはひび割れを考慮するトリリニアのふたつが一般的で、Cloughモデルはこのバイリニア型となっていますが、 IMKモデルはさらに最大強度到達を考慮したトリリニア型、オールマイティモデルはnリニア型(nは7まで)を採用しています。

　再載荷時の指向点を決める規則には主に、それまでの最大到達変位を目指す最大点指向型、最大到達変位を目指すものの再載荷のはじめは強度が出ないピンチング/スリップ型、 また除荷剛性のまま拡大していくバイリニア型などがあります。今回取り上げた３つの復元力特性はどれも最大点指向型を基本としていますが、 IMKモデルとオールマイティモデルはピンチング型とすることも可能で、ピンチング形状を膨らませることでバイリニア型を模すことも可能です。

　耐力低下は、既存の復元力特性では再載荷時であろうと骨格曲線まで強度が上昇するはずですが、実際にはそれより手前に落ち込む現象を表現するために、考えられたもので いくつかその方法が提案されていますが、今回は、載荷曲線が領域を囲んでエネルギーを吸収することで、骨格曲線を内側に後退させるIMKモデル型のものを用いています。 Cloughモデルにはこの耐力低下はなく、IMKモデルでは4つ、オールマイティモデルでは2つのパラメータを与えています。

Description

　Change GMで地震動を、右側のスライダかもしくは入力欄で入力倍率を選択してRunで揺らしてみましょう。
　右側のResultにCollapseと出たら壊れてしまったことを示します。Not Yetと出たらその倍率では壊れなかったわけです。
　崩壊に至る最小の倍率が崩壊倍率だと言えるでしょう。さて、別の地震動でも崩壊倍率は同じになるでしょうか。

Next:漸増動的解析

　自分でいくつか試してみて崩壊倍率を探してもらったと思いますが、これをコンピュータにやってもらえばだいぶ楽に、多くの地震波について崩壊倍率を 得ることが出来るはずです。これが漸増動的解析、通称IDAです。

NoGM*MCA*StD*
Description

　スライダで入力地震波の数を選択します、一般的なCPUの性能では1波2秒くらいで出来るはずです。

　ここでは横軸に最大変位を、縦軸に入力倍率を取っています。先ほどは崩壊の定義を詳しく示しませんでしたが、 少なくともここでは少しの倍率の増加で変位が急激に拡大することを崩壊とみなすことが出来ます。

　さて、耐力低下も最大強度到達もないCloughモデルを用いても、この崩壊は明瞭なものになるでしょうか。

*NoGM=Number of Used Ground Motions, *MCA=Mean Collapse Amplitude, *StD=Standard Deviation
